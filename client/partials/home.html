SKIP - The System of Kanji Indexing by Patterns

Introduction

The System of Kanji Indexing by Patterns (SKIP) is a system for ordering kanji. Developed by Jack Halpern and used as the main index for kanji in the New Japanese-English Character Dictionary (Kenkyusha/NTC, 1990) and other dictionaries published by the Kanji Dictionary Publishing Society.

Why use SKIP?

The system is very convenient because it can be learned quickly, easy to use, and requires little prior knowledge of kanji. Instead of organizing kanji based on radicals, they are categorized by pattern.

Determining SKIP codes

A SKIP code is made up of three numbers (e.g. "1-4-3"). The first number indicates the visual geometry of the kanji character. The remaining two numbers represent the number of strokes.

Visual style and the first SKIP number

The SKIP system groups kanji based on how the character can be divided into sections. There are four main categories:

  1. Left-Right: Left-Right kanji can be broken down into vertical sections.
  2. Up-Down: Up-Down kanji can be broken down into horizontal sections.
  3. Enclosure: Enclosure kanji have one element surrounding other elements.
  4. Solid: Solid kanji cannot be broken down into elements based on the other three categories. These kanji are usually formed from a single radical. Solid kanji are broken down into four sub-patterns:
    1. Top line - Prominent horizontal stroke at the top
    2. Bottom line - Prominent horizontal stroke at the bottom
    3. Through line - Prominent vertical stroke through the middle
    4. Others - Kanji that don't fall into the preceding patterns
The first number in a SKIP code indicates which category the kanji character belongs in.

Stroke counts for styles #1, #2, and #3

For styles 1, 2, 3, the number of strokes in the left/upper/enclosing section will be the second number in the SKIP code. The number of strokes in the right/lower/internal section will be the third number in the SKIP code.

Stroke counts for style #4

For style 4, the number of strokes in the character will be the second number in the SKIP code. The sub-pattern number will be the third number in the SKIP code.
